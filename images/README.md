# Carpeta de Imágenes - El Roperito

## Estructura recomendada

```
images/
├── productos/          # Fotos de productos
├── banners/           # Imágenes para banners y hero sections
├── logos/             # Logotipos y variantes
├── icons/             # Iconos personalizados
└── categorias/        # Imágenes de categorías
```

## Formatos recomendados
- **Productos**: JPG (alta calidad) o WebP
- **Banners**: JPG o PNG
- **Logos**: SVG (vectorial) o PNG con transparencia
- **Iconos**: SVG o PNG

## Nomenclatura sugerida
- `producto_[id]_[descripcion].jpg`
- `banner_[seccion]_[nombre].jpg`
- `logo_[variante].svg`
- `icon_[nombre].svg`

## Optimización
- Comprimir imágenes para web
- Usar formatos modernos (WebP, AVIF)
- Mantener relaciones de aspecto consistentes
- Tamaños máximos recomendados: 1200px para productos, 1920px para banners
